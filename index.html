A propos de moi

Mes projets

Projet étudiant au Cuej : Nos amis les bêtes

Ce projet étudiant impliquait la création d'un site d'articles pour le Centre Universitaire d'Enseignement du Journalisme (CUEJ), permettant aux étudiants de publier des articles en utilisant un CMS conçu par l'équipe. Les technologies utilisées étaient HTML/CSS, JavaScript, PHP et MySQL, avec PHP et MySQL pour la gestion des données et la base de données pour stocker les articles. HTML/CSS ont été utilisés pour la structure et l'apparence du site, tandis que JavaScript a été utilisé pour ajouter des fonctionnalités interactives. Le résultat final était un site professionnel, fonctionnel et facile à utiliser, avec un CMS permettant aux étudiants de publier et de gérer leurs articles en temps réel. Mon rôle dans le projet était toute la partie intégration et une partie du back-end.

Projet étudiant SAE 203 : BestManga

Ce projet étudiant avait pour but de développer un site web avec des articles captivants et des animations CSS pour améliorer l'expérience utilisateur. Le site incluait une base de données complexe avec des relations entre les tables pour une gestion efficace. Les technologies utilisées étaient HTML/CSS pour la structure et la présentation du site, PHP pour la gestion de la base de données et la génération de pages en temps réel, MySQL pour le stockage de données, et Twig pour faciliter la gestion des éléments du site. Ce projet m'z permis de développer mes compétences en matière de développement web en travaillant avec des technologies modernes pour construire un site dynamique et fonctionnel.
